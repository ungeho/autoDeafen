# autoDeafen for League of Legends

## 📌 概要
League of Legends で **試合開始を検知し、自動で `/deafen` を1度だけ送信してミュート状態にするツール**です。  
試合開始直後のチャットミュート操作を自動化し、ゲームに集中しやすい環境を提供します。

---

## 🎯 目的
- 試合開始後すぐに `/deafen` を実行したい
- チャットやピンによるストレスや情報ノイズを減らしたい
- ゲームの開始にコマンドを入力する手間を減らしたい

---

## ⚠ 注意事項（重要）
このツールは **実際にキーボード入力を送信**する形で動作します。  
そのため以下の条件を満たす必要があります：

| 項目 | 必要条件 |
|---|---|
| フォーカス | LoLクライアントがアクティブになっていること |
| 入力モード | IMEが **半角英数** であること |
| 試合開始直後の操作 | 開始後2秒程度はショップ・設定・チャットを開かない |

※ 条件が満たされない場合、正しくコマンドが入力されないことがあります。

---

## 🛠 インストール手順

### 1. Pythonをインストール  
https://www.python.org/downloads/

インストール時に **✔ Add Python to PATH** を必ず有効にする。

### 2. 必要パッケージをインストール

```bash
pip install requests pynput urllib3
```

### 3. リポジトリの取得

```bash
git clone https://github.com/<yourname>/autoDeafen.git
cd autoDeafen
```

### 4. 実行

```bash
python auto_deafen.py
```

## 動作

実行すると以下の動作を行います：
1. LoLのライブAPIを監視
2. 試合開始のイベントを検出
3. 指定秒数後(デフォルトは1秒)に`/deafen`をチャットへ**一度だけ**送信
5. 試合終了を検知するとライブAPIを監視する状態へ戻り、次のゲームでも同様に動作
